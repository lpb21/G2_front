# Version de Node

    Node: 18.12

# Instalacion de Dependencia

    Utilizar el siguiente comando `npm i` or `npm install`


# Start de Front-End
* frontend
* Start ReactJS frontend en el puerto `http://localhost:3000` con el comando:
    ``` 
    npm run start 
    ```
    Utilizar el siguiente comando `npm run start`
